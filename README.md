# Sales Tracker App

A React + TypeScript + Vite application for tracking sales and customers using Supabase.

## Colleague Onboarding & Setup

To get started with this project on your local machine, follow these steps:

### 1. Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### 2. Basic Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/buyung-alexion/https-github-com-Buyung-sales-tracker-app.git
   cd "Sales Tracker"
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```

### 3. Environment Variables
This project requires connection keys for Supabase.
1. Create a file named `.env.local` in the root directory.
2. Copy the contents of `.env.example` into `.env.local`.
3. Ask the project owner for the specific `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` values and paste them in.

### 4. Running the App
Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.
